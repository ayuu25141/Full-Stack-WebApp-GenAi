# FormGen AI

Full-stack application with separate frontend and backend

## Project Structure
```
project-root/
├── front/          # Frontend application
└── back/           # Backend API
```

## Setup

### Frontend Setup
```bash
cd front
npm install
cp .env.example .env  # Edit with your values
npm start
```

### Backend Setup
```bash
cd back
npm install
cp .env.example .env  # Edit with your values
npm start
```
